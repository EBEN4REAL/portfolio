My Portfolio Some Of My Works

Epump - Fuel station automation portal

Epump improves the fuel stations' ability to monitor, control everything that happens in their fuel station remotely in real time and grow their business while reducing theft, fraud and revenue leakage. This solution is being used by over 5,000 fuel stations in Africa including Nigeria and Kenya.

Clearance Portal

Designed and built an automated clearance portal for Covenant University currently being utilized by over 12k students on campus. This portal facilitates students' change of program and also helps student perform clearance online without having to visit the University

Github clone

Designed a clone of github website as a fun side project, where a github username can be entered and information pertaining to the username would be displayed including list of repositories, followers, following and other relevant details associated with the user using React, Typescript and GraphQL

YTS - Movies listings app clone

Designed a clone of the popular movies download app called YTS which render popular movies including the details including imdb rating. This was built using react, typescript, node/express & mongoDB
