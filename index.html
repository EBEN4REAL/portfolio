My Portfolio Some Of My Works

Epump - Fuel station automation solution

Developed Epump, a fuel station automation portal that improves the fuel stations' ability to monitor, control everything that happens in their fuel station remotely in real time and helps stations grow their business while reducing theft, fraud and revenue leakage. This product is being used by ~30k fuel stations in Africa. Tech stack: Vue, JavaScript, Sass & Bootstrap

Coding playground very similar to Leetcode app

Designed and developed a coding playground very similar to leetcode which can be used by people to enhance their programming skills by solving coding questions.

Double-dice betting app

Collaborated with a team of 3 developers to build Double-dice, a web3 based web app that was designed and built for users to place bets using their crypto wallet. Metamask was the main testing tool used in during this project. Tech stack: React, web3.js, Typescript, GraphQL

Clearance Portal

Built an automated clearance portal for Covenant University currently being utilized by over ~12k students on campus. This portal facilitates students' change of program and also helps student perform clearance online without having to visit the University. Tech stack: PHP, MySQL, OOP, JavaScript and Ajax

Github clone

Designed a clone of github app as a fun side project, where a github username can be entered and information pertaining to the username would be displayed including list of repositories, followers, following and other relevant details associated with the user. Tech stack: React, Typescript & GraphQL

YTS - Movies listings app clone

Designed a clone of the popular movies download app called YTS which renders popular movies including details such as imdb rating leveraging Open movie database (OMDB) API. Tech stack: react, typescript, node/express & mongoDB.

Smart energy management solution

Designed and developed a smart meter management solution for individuals and corporate bodies and government parastatals to easily manage their smart meters and track their meter consumption and revenue. Tech stack: React, Redux and Tailwind
